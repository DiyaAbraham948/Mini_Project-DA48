# Women Crime Analysis India

# Project Overview
This project focuses on analyzing crimes against women in India from 2001–2022 using NCRB crime data.  
The dataset was cleaned, transformed, and visualized using Excel and Power BI to identify crime trends, high-crime states, and regional crime distribution patterns.
# Data Source
This dataset was obtained from Mendeley Data.
# Dataset Name
Crime Against Women Dataset for Dark Spot Identification in India
# Source Link
https://data.mendeley.com/datasets/whrdh8c5zb/1/files/f63fe570-5fe5-4efe-a229-73dbc83205a7 
# Original Publisher
National Crime Records Bureau (NCRB) & Census of India
# Description
The dataset contains 21 years of NCRB crime statistics and census parameters across districts in India.
# Dataset Contents
- Dataset.csv – Raw uncleaned dataset
- Cleaned Excel Dataset
- Power BI Dashboard File (.pbix)
# Tools Used
- Microsoft Excel
- Microsoft Power BI
- GitHub
# Methodology
# 1. Data Cleaning in Excel
The dataset was cleaned and transformed using Microsoft Excel.
# Cleaning Steps Performed
- Removed blank rows and unnecessary columns
- Renamed lengthy column names into shorter readable names
- Corrected data types
- Converted numerical values into whole numbers
- Organized state-wise and year-wise data
- Removed unwanted and duplicate fields
# Columns Removed
Some columns were removed to reduce dataset size and improve Power BI loading performance because the original dataset was too large to load efficiently.
Removed columns include:
- Custodial Rape
- Other Rape
- Kidnapping and Abduction of Wife and Girls
- Kidnapping and Abduction of Others
- Insult to Modesty of Women
- Importation of Girls.
# 2. Data Visualization using Power BI
Interactive dashboards were created using Power BI to analyze crime patterns across India.
 Dashboard Features
- KPI Cards for Total Crimes, Murder, Rape, and Kidnapping
- State-wise Crime Analysis
- Crime Distribution Charts
- Crime Trend Over Years
- Interactive State and Year Slicers
- Crime Intensity Map Visualization
- Key Insights Page
# Final Dataset Columns Used
- State/UT
- District
- Year
- Murder
- Attempt to Murder
- Rape
- Kidnapping and Abduction
- Dowry Deaths
- Assault on Women
- Cruelty by Husband or Relatives
- Total Crimes
 # Key Insights
- Uttar Pradesh recorded one of the highest crime rates among analyzed states.
- Crimes such as Rape, Murder, and Kidnapping contributed significantly to total crimes against women.
- Crime distribution varied across different states and regions of India.
- Interactive filters enabled state-wise and year-wise analysis.
- Map visualizations helped identify high-crime regions.
# Limitation
The dataset represents only crimes officially reported to the police and may not reflect the actual number of incidents.
